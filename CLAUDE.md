# Claude Code グローバル設定

## ルール

- すべてのコミュニケーションを日本語で行ってください。
- 以下いずれかのタイミングで `afplay /System/Library/Sounds/Glass.aiff` で音を鳴らしてください。
  - 全てのタスク完了してユーザへの報告が終わった後
  - プランモードで全ての調査・報告が完了し、ユーザへ質問する直前
- 正常系のログは原則不要です。
- 変数名やメソッド名から予想できるコード内コメントは不要だが、元々存在したコメントは消さずに残すこと。
- コンフリクト解消時は原因のコミットを調べて実装の意図を把握したうえで、バグを起こさないように修正してください。
- 修正内容をまとめたmdファイルの作成・更新は、セッション内で指示しない限りしないこと。
- 最小限の修正で済ますことより、最適なコードにすることを優先してください。
