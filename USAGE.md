# 使い方

## 画面構成

```
┌─────────────────────────────────┬──────────────────┐
│ エマージェンシーボタン履歴          │  怪しさチャート    │
├──────────────────────────────── │                  │
│ [プレイヤー数] [クールタイム] [タイマー]│                  │
│ [マップ変更] [全リセット] [名前登録]  │                  │
│ [経路ボタン]  [終了] [ラウンド]      ├──────────────────┤
│                                 │  死体チャート      │
│           マップ                  │                  │
│                                 │                  │
└─────────────────────────────────┴──────────────────┘
```

## 初期設定

`assets/config.json` を編集してプレイヤー名と自分のカラーを設定します。

```json
{
  "players": [
    { "color": "red",  "name": "名前" },
    { "color": "cyan", "name": "自分の名前", "isMyself": true }
  ]
}
```

- `name` が空のプレイヤーはアプリに表示されません
- `isMyself: true` を付けたプレイヤーが自分として扱われ、怪しさチャートから除外されます
- アプリ起動後でも **名前登録** ボタンから変更可能です

## 各機能の使い方

### プレイヤー登録

**名前登録** ボタンを押すと名前登録画面が開きます。

- 名前を入力するとプレイヤーがアクティブになり、空にすると非表示になります
- 「自分」のカラーをここで変更できます

---

### マップ操作

- **マップ変更** ボタン：5種類のマップ（The Skeld / MIRA HQ / Polus / The Airship / Fungle）から選択します
- **プレイヤーアイコンのドラッグ**：マップ上の任意の位置にドラッグして位置を更新します
- **プレイヤーアイコンのクリック**：プレイヤーを選択状態にし、ステータス変更UIを表示します（再クリックで閉じます）

---

### ステータス変更

プレイヤーアイコンをクリックすると表示されるUIで操作します。

| 操作 | 内容 |
|------|------|
| キル済みアイコン | 現在のラウンドでキル済みに変更（頭骨マークが付く） |
| 追放アイコン | 現在のラウンドで追放済みに変更（×マークが付く） |
| -10 / -1 / +1 / +10 | 主観的怪しさスコアを調整（中央の数字がスコア） |

スコアは怪しさチャートの計算に反映されます。

---

### 移動経路の記録

1. プレイヤーアイコンをクリックして選択状態にします
2. マップ上でドラッグすると選択プレイヤーの色で経路が描かれます

経路コントローラーのボタン：

| ボタン | 内容 |
|--------|------|
| ゴミ箱 | 選択中プレイヤーの経路を全消去 |
| 左矢印 | アンドゥ（1本前の線を消す） |
| 右矢印 | リドゥ（消した線を戻す） |

---

### キルタイマー

画面上部右側の秒数表示エリアで操作します。

| 操作 | 内容 |
|------|------|
| 長押し | タイマー開始（またはリスタート） |
| 動作中・左タップ | -5秒 |
| 動作中・右タップ | +5秒 |
| 停止中にタップ | 過去のキルタイマー履歴を表示 |

背景色でキル可能回数を視覚的に確認できます：

- **白**：キル不可
- **黄**：1回キル可能
- **赤**：2回以上キル可能

---

### エマージェンシーボタン履歴

画面左上のボタン履歴エリアで、ボタンを使った順番を記録します。

- ボタンアイコン（赤いボタン）をクリック → そのプレイヤーの使用順を記録
- 番号をクリック → 記録を取り消し（後の番号も自動で更新されます）

死亡済みのプレイヤーには × が表示され、操作できません。

---

### クールタイム設定

画面上部中央でスクロールして調整します。

| 項目 | デフォルト | 範囲 |
|------|-----------|------|
| ボタン | 15秒 | 10〜60秒 |
| キル | 30秒 | 10〜60秒 |

クールタイムの値はキルタイマーの背景色判定に使用されます。

---

### ラウンド管理

- **ラウンド：N** の表示にカーソルを合わせるとラウンド一覧が展開します
- 番号をクリックするとそのラウンドの状態（プレイヤー位置・経路）を表示します
- プレイヤーを移動したりステータスを変更すると、最終ラウンドが自動更新されます
- **全リセット** ボタンで全プレイヤーの位置・死亡状況・経路をリセットします

---

### 怪しさチャート（右上）

生存しているプレイヤーを怪しさスコア順に並べて表示します。

- **黒**：最も怪しい（筆頭容疑者）
- **グレー〜薄黒**：怪しさに応じたグラデーション
- **白**：白確定（自分目線でシロ）

スコアは「死体ごとの容疑者設定」と「主観的スコア」から自動計算されます。

クリックするとウィンドウが元の位置に戻ります（縮小後の復帰操作）。

---

### 死体チャート（右下）

キルされたプレイヤーごとに、発生ラウンドと容疑者を表示します。

| 列 | 内容 |
|----|------|
| 左端 | キルされたプレイヤー |
| 中央の数字 | キルされたラウンド（クリックでそのラウンドに移動） |
| 右側のアイコン群 | 容疑者リスト |

容疑者アイコンの操作：
- **黒背景**：容疑者（怪しさスコアにマイナス補正がかかる）
- **白背景**：白確定（スコア計算から除外）
- クリックで黒↔白を切り替えます

---

### ウィンドウ操作（AmongUsとの切替）

- カーソルがアプリウィンドウの外に出ると自動的にウィンドウが縮小し、AmongUsにフォーカスが移ります
- ウィンドウ縮小後、怪しさチャート（右上エリア）をクリックするとウィンドウが元の位置に戻ります
